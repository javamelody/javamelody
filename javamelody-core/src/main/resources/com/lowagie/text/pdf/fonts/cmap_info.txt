Note from evernat:
The files in this directory are extracted, in order to have only Chinese fonts, from iTextAsian.jar
which can be downloaded from http://sourceforge.net/projects/itext/files/extrajars/ExtraJars1.0/iTextAsian.jar/download
(iTextAsian from iText v5+/ExtraJars v2 is not a compatible jar for iText v2.1.7)

Note from iText:
The *.cmap and *.properties files in this directory are necessary
to produce PDF files with iText that use CJK fonts.

Note that whatever value you pass for the 'embedded' parameter
with the method BaseFont.createFont, the font WILL NOT BE embedded.

These specific metrics files were created by Paulo Soares and
may be used, copied, and distributed for any purpose and without
